    Wattanapat-clone – Wattanapat Hospital Trang (Clone Project) โรงพยาบาลวัฒนแพทย์ ตรัง -clone

    This project is a clone of the Wattanapat Hospital Trang website, developed using the Laravel framework. It focuses on replicating key features of the original hospital site, including patient registration and health checkup reservation functionality. The backend is powered by Laravel, leveraging MVC architecture and Eloquent ORM for database management. On the frontend, the project uses jQuery and AJAX to enable dynamic form interactions and seamless user experience without full-page reloads. The system includes: A multi-step reservation form with validation RESTful API integration for health checkup registrations Logging and error handling mechanisms for debugging Support for JSON input and response for API clients like Postman This clone is intended as a learning and development project to improve skills in modern web application development using Laravel and interactive frontend technologies.
