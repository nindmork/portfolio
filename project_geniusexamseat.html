    • Genius Exam Seat - ระบบจัดสรรที่นั่งสอบอัจฉริยะ

    GENIUS EXAM SEATS Application ถูกพัฒนาขึ้นเพื่อเป็นโซลูชันที่ครบวงจรสำหรับการจัดสรรที่นั่งสอบและการจัดการอย่างอัจฉริยะในสถานศึกษาต่างๆ ด้วยระบบนี้, การจัดที่นั่งสอบไม่เพียงแต่เป็นไปอย่างเป็นระเบียบและครอบคลุม แต่ยังเต็มไปด้วยความยืดหยุ่นและประสิทธิภาพสูงสุด

    ฟีเจอร์การทำงานหลัก

  • คำนวนและจัดสรรที่นั่งสอบที่ซับซ้อน
  • คำนวนและจัดสรรห้องสอบ
  • Drag and Drop เพื่อปรับแต่งตำแหน่งที่นั่ง
  • ระบบเพิ่มข้อมูลจำนวนด้วยการอัพโหลดไฟล์ .xlsx
  • Repost ต่างๆ เช่น ตารางสอบรายบุคคลสำหรับนักศึกษา และรายงานอื่นๆ


  • พัฒนา Frontend ด้วย Typescript NextJs, Redux, Tailwind, jQuery
  • พัฒนา Authentication ด้วย AWS, JWT Token ,GoogleOAuth2, FacebookOAuth2
  • พัฒนา Backend ด้วย Djanog, REST Framework, postgreSQL, jQuery,
